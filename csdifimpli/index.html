9) CSS Combinators

A combinator is something that explains the relationship between the selectors.
A CSS selector can contain more than one simple selector. Between the simple selectors, we can include a combinator.

There are four different combinators in CSS:

i) Descendant selector (space)
ii) Child selector (>)
iii) Adjacent sibling selector (+)
iv) General sibling selector (~)

10) Pseudo selectors in css

The Pseudo code selector is always specifies a special state of the selected element.

Ex.
i) :active - Selects the active link
ii) :checked - Selects every checked input element
iii) :hover - to change button color when user pointr hover on it.
